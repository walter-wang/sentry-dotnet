    Write(LogEventInfo)

    If the event level >= the MinimumEventLevel, the logEvent is captured as an event by sentry.

    If the event level is >= the MinimumBreadcrumbLevel, the event is added as a breadcrumb to the Sentry Sdk.

    If sentry is not enabled, this is a No-op.
